Documentation: ZKAPREKAR_CALC

Kaprekar's Constant Routine

Purpose:

This program demonstrates the Kaprekar routine for 4-digit numbers, which converges to the constant 6174.

Algorithm:
  1. Take any 4-digit number with at least two different digits
  2. Arrange the digits in descending order
  3. Arrange the digits in ascending order
  4. Subtract the smaller number from the larger
  5. Repeat with the result
Mathematical Background:

Kaprekar's constant, discovered by Indian mathematician D.R. Kaprekar in 1949, is 6174. For most 4-digit numbers, the routine converges to 6174 in at most 7 iterations.

Example:
Starting with: 1234
4321 - 1234 = 3087
8730 - 0378 = 8352
8532 - 2358 = 6174

Converged in 3 iterations!
Input Parameters:
  • Input Number: A 4-digit number (leading zeros allowed)
  • Max Iterations: Maximum iterations before stopping (default: 20)
Output:

The program displays each iteration showing the descending arrangement, ascending arrangement, and their difference until convergence to 6174 is achieved.

Special Cases:
  • Numbers with all identical digits (e.g., 1111) are not valid
  • Some numbers may reach 0 (repdigits) and cannot converge
  • The constant 6174 is a fixed point: 7641 - 1467 = 6174

VA01 - Create Sales Order

Purpose:

Use this transaction to create a new sales order in the system. A sales order is a contractual agreement between your organization and a customer for the delivery of goods or services.

Procedure:
  1. Enter the Order Type (e.g., OR for standard order)
  2. Enter Sales Organization, Distribution Channel, and Division
  3. Enter the Sold-To Party (customer number)
  4. Optional: Enter PO Number and PO Date
  5. Add line items with material numbers and quantities
  6. Review header data, conditions, and shipping information
  7. Save the order (Ctrl+S or Save button)
Navigation:
  • F3 - Back
  • Shift+F3 - Exit
  • F12 - Cancel
  • Ctrl+S - Save
Tabs:
  • Sales: Overview of all items in the order
  • Item Detail: Detailed information for selected item
  • Ordering Party: Partner functions and contact information
  • Procurement: Purchasing and vendor data
  • Shipping: Delivery dates and shipping conditions

ZKAP - Kaprekar Constant Calculator

Purpose:

This program demonstrates the Kaprekar routine, which converges to the constant 6174 for most 4-digit numbers.

How to Use:
  1. Enter a 4-digit number (with at least two different digits)
  2. Press F8 (Execute) or click the Execute button
  3. View the iteration results showing convergence to 6174
Special Keys:
  • F8 - Execute the program
  • F3 - Back to previous screen
Tip:

Press F1 at any time to get context-sensitive help for the current field or screen.

SAP Glossary

Common SAP Terms

ABAP

Advanced Business Application Programming - SAP's proprietary programming language

ATP

Available-to-Promise - Check if products are available for delivery

BAPI

Business Application Programming Interface - Standard interfaces for business objects

Client

Independent business unit within an SAP system (e.g., 020)

Distribution Channel

Route through which products reach customers

Division

Product grouping for sales and distribution

F4 Help

Search help/value help accessed by pressing F4 or clicking the search icon

Incoterms

International Commercial Terms defining delivery responsibilities

Material

Products or services that can be procured or sold

Plant

Physical location where materials are produced or stored

Sales Order

Contract between company and customer for delivery of products/services

Sales Organization

Organizational unit responsible for sale and distribution of products

Sold-To Party

Customer who orders the goods or services

Ship-To Party

Customer who receives the goods

T-Code

Transaction Code - Shortcut to access SAP transactions (e.g., VA01)

Table Control

Grid display for entering multiple line items
